# Contributing Guide

Loving father and want to get involved? Thanks!

## Set up

```bash
# Clone this repo
$ git clone git@github.com:umijs/father-next.git
$ cd father-next

# Install Dependencies
$ pnpm i

# Compile Src
$ pnpm build
```

## Run Example

```bash
$ cd examples/normal
$ pnpm build
```

## Test

```bash
$ pnpm test
```

## Release

```bash
$ pnpm release
```
